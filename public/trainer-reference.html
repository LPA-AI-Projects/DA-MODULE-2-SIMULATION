1. Simulation Overview

This simulation is a four-topic recap of Module 2 with 5 questions per topic (20 total). Learners work through realistic Gulf Freight Co. HR data scenarios. This guide provides facilitation notes, answer keys, and debrief guidance.

Scenario given to learners: Gulf Freight Co. needs an HR dashboard that tracks headcount, turnover, and Emiratisation across four departments — and automatically flags departments that need attention.

Benchmarks used throughout:

  • Turnover flag threshold: above 5%
  • Emiratisation flag threshold: below 20%
  • Flag rule (combined): flag if turnover > 5% OR Emiratisation < 20%

Scoring logic (unchanged in live simulation):

  • 20 questions × 1 point max each = 20 points maximum
  • Correct on first attempt → 1 point
  • Correct on second attempt → 0.5 points
  • Two wrong attempts → 0 points, correct answer revealed
  • LMS score = (total points ÷ 20) × 100, rounded

7. Facilitation Tips

  • Data Cleaning: Distinguish "untidy" from "functionally broken" — only text-as-numbers and blanks break formulas silently.
  • XLOOKUP: #N/A means no exact match — check spelling, spaces, and column references.
  • PivotTables: Train learners to audit conditional formatting against the stated rule cell by cell.
  • IF Logic: Say the rule aloud with "OR" before writing formula — most common mistake is using AND.
